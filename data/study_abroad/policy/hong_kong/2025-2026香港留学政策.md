# 2025-2026年香港留学政策指南

## 一、IANG签证政策（非本地毕业生留港/回港就业安排）

### 1.1 核心政策变化（2026年）

| 维度 | 旧政策（2025） | 2026新政 | 变化 |
|------|---------------|----------|------|
| IANG首次逗留期 | 1年 | **2年（24个月）** | ✅ 翻倍 |
| 续签申请窗口 | 到期前4周 | **到期前3个月（90天）** | ✅ 放宽7倍 |
| 非本地生限额 | 40% | **50%（2026/27学年起）** | ✅ 扩招 |
| 高才通合资格校 | 185所 | **198所** | ✅ +13所 |
| 大湾区港校IANG | 试行中 | **试行期再延2年** | ✅ 延续 |

### 1.2 应届非本地毕业生（毕业6个月内申请）

| 项目 | 详情 |
|------|------|
| 是否需要Offer？ | ❌ 不需要！毕业后6个月内直接申请，无雇主也能批 |
| 首次获批时长 | **24个月（2年）**，无条件逗留，自由换工作/创业 |
| 申请费 | 600港币 + 1300港币签证签发费 = **1900港币** |
| 审批时间 | 收到材料后约2周（6-8月高峰期可能延长） |
| 获批率 | **96%+**（2025年数据：28,710人申请，27,630人获批） |
| 能否带家属？ | ✅ 配偶+18岁以下子女可申请受养人签证，配偶可在港合法工作 |

### 1.3 非应届毕业生（毕业超过6个月后申请）

| 项目 | 详情 |
|------|------|
| 是否需要Offer？ | ✅ 必须有！需已获聘用 |
| 工作要求 | 职位须由学位持有人担任，薪酬福利达市场水平（硕士建议≥2万港币/月） |
| 首次获批时长 | **24个月（2年）** |
| 审批时间 | 约4周（可加急5个工作日，+1200港币） |

### 1.4 大湾区校园毕业生（港中深/港科广/北师港浸大/港城大东莞）

| 项目 | 详情 |
|------|------|
| 能否申IANG？ | ✅ 可以！试行期再延2年 |
| 应届生 | 毕业6个月内申请，无需Offer，首次批24个月 |
| 非应届生 | 需有Offer，薪酬达市场水平，首次批24个月 |

### 1.5 STEM方向毕业生优待
- 计算机科学、人工智能、数据科学、电子工程等STEM方向的港校毕业生
- 2026年起首次IANG签证可以直接批**3年**
- 加上额外1年的求职缓冲期，前后最长可以拿到4年的合法留港时间

## 二、续签政策（2026年3月1日起）

### 2.1 续签规则大放宽

| 对比项 | 旧规则 | 2026新规 |
|--------|--------|----------|
| 续签申请时间 | 到期前4周 | 到期前3个月（90天） |
| 断签风险 | 很高（材料不齐就断签） | ✅ 几乎消除（90天准备期） |
| 续签材料 | 雇主证明+公司文件 | 同左，但时间充裕 |
| 续签模式 | 2+3+3 | ✅ 2+3+3（IANG首次2年→续签3年→再续签3年→满7年永居） |

### 2.2 续签具体要求
- 必须在港受雇，工作与学历相关
- 薪酬达标：硕士普遍要求≥2万港币/月
- 居住证明：有住址、纳税、强积金等"通常居住"证据
- 无行业限制：可自由换工作、换雇主
- 可创业：自雇也算，需提供业务证明文件

## 三、永居政策

### 3.1 永居申请核心要求
- **累计在港连续逗留满7年**（从首次入境香港的"小白条"日期开始计算）
- 求学期间时间全部计入7年永居时间
- 7年期间需保持合法居留状态（无非法打工、逾期滞留等记录）
- 能提供持续在港的证明材料（工作证明、税单、银行账单、电话费账单等）

### 3.2 永居申请流程
1. 递交申请：填写ID991表格，提交相关证明材料
2. 审核：入境处审核申请材料，可能要求补充文件
3. 获批：审核通过后领取永久居民身份证

## 四、学生签证政策

### 4.1 申请条件
- 获得香港认可院校的录取通知书
- 具备足够的经济能力支付学费和生活费
- 无犯罪记录
- 身体健康

### 4.2 有效期
- 学生签证有效期通常与课程时长挂钩
- 最短6个月，最长可达4年

### 4.3 受养人签证
- 全日制学位课程学生的配偶及18岁以下子女可申请受养人签证
- 配偶可在港合法工作
- 子女可在港就读学校
