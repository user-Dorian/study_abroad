# 2026北上广深留学生回国落户政策全汇总

## 一、北京

### 落户要求

1. **学历**：境外硕士及以上
2. **境外时长**：累计满360天（出入境记录严格算，回国天数不计）
3. **年龄**：≤45周岁（博士可放宽到50）
4. **申请时效**：回国2年内由单位提交
5. **工作与社保**：入职有落户指标的在京单位（央企/国企/高新企业/事业单位为主）；连续缴纳3个月北京社保且申请时在缴
6. **劳动关系**：出国前已与原单位解除

### 落户福利

1. **购房/购车**：直接获得北京购房资格、摇号资格
2. **子女教育**：按京户就近入学，部分区域重点校倾斜
3. **人才补贴（区为主）**：
   - 海淀/朝阳：人才公寓（市场价约50%），最长3年
   - 经开区：百强院校入职满1年，本科1万/年、硕士3万/年、博士5万/年，连发3年
4. **创业支持**：中关村"科创海英"计划，TOP200博士有柔性通道

### 注意事项

- 必须通过工作单位提交申请，不能个人提交
- 疫情期间国内上网课时间可计入365天时长
- 对用人单位资质要求较高，通常要求有中央或北京的落户指标

## 二、上海（2026.2新政）

### 落户要求

1. **学历**：境外本科及以上
2. **年龄**：距退休≥5年（一般≤55）
3. **院校梯度（重点）**：
   - QS/THE/ARWU任一TOP50：直接落户，零社保
   - TOP51-100：连续6个月社保可落户
   - 普通硕士：1倍社保基数、6个月可落户
   - 普通本科：1.5倍社保基数、12个月可落户
4. **特殊通道**：
   - 科创清单企业+紧缺专业：6个月社保可落
   - 创业（持股≥30%、实缴≥50万）：6个月社保可落
5. **家属**：配偶+未成年子女可随迁

### 落户福利

1. **现金补贴（浦江人才计划）**：10-30万（依行业、项目不等）
2. **租房/住房**：
   - 部分区：500-1500元/月租房补贴，1-3年
   - 人才公寓优先配租
3. **子女教育**：居住证/海外人才证可就近入学，国际学校资源丰富
4. **创业支持**：创业园入驻、项目资助、贴息贷款等

### 政策有效期

《留学回国人员申办上海常住户口实施细则》已于2025年12月1日正式施行，有效期至2030年11月30日。

## 三、广州

### 落户要求

1. **学历及年龄**：
   - 博士：≤50岁
   - 硕士：≤45岁
   - 本科（境外）：≤40岁
2. **社保**：广州1个月参保记录即可（在缴）
3. **工作/创业**：在穗就业（签劳动合同）或合法创业
4. **其他**：境外时长、院校排名、回国时间无硬性限制

### 落户福利

1. **人才补贴（市+区）**：
   - 市级：本科1.5万、硕士2.5万、博士10万（多区同标准）
   - 从化/增城等：额外住房补贴（以当年公告为准）
2. **购房**：直接获得广州购房资格，无社保年限要求
3. **子女教育**：就近入学，公办学位充足
4. **创业**：最高50万无息贷款、场地补贴、税费减免

### 各区差异化补贴

- 黄埔区：本科海归提供2万元补贴
- 番禺区：租房补贴每月1500元（约24个月）

## 四、深圳

### 落户要求

1. **学历&年龄**：
   - 本科≤35周岁
   - 硕士≤40周岁
   - 博士≤45周岁
2. **社保**：缴纳1个月社保即可办理，部分区域可放宽
3. **流程**：最快24分钟办结，全流程网办深圳政府在线
4. **其他**：无境外停留时长、院校排名硬门槛，包容性极强

### 落户福利

1. **市级人才补贴（名校梯度）**：
   - 本科：可达3万元左右
   - 硕士：可达5万元左右
   - 博士：可达10万元
2. **区级加码**：
   - 龙岗区：世界排名前100的硕士毕业生，一次性领3万元
   - 龙华区：QS前300，本科3万、硕士5万、博士8万
   - 南山区：四大排名Top 150院校，本科达万、硕士达5万
3. **住房**：为青年人才提供租金为市场价60%的过渡性住房

### 政策优势

- 全流程网办，效率极高
- 落户门槛最低，包容性最强
- 补贴力度大，真金白银吸引人才

## 六、落户申请流程

1. **学历认证**：登录教育部留学服务中心网站，进行国外学历学位认证
2. **准备材料**：学历认证报告、护照及出入境记录、身份证、户口本、劳动合同、社保缴纳证明等
3. **单位申请**：由用人单位登录当地人社局网站提交申请
4. **审核公示**：等待审核，审核通过后进行公示
5. **办理落户**：公示无异议后，前往派出所办理户口迁移
