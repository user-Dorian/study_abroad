## 一、本科申请条件

### 申请路径

#### 高考赛道（占比66%）

| 院校 | 高考要求 | 英语要求 |
|---|---|---|
| 香港大学 | 超一本线130分 | 130分以上 |
| 香港中文大学 | 各省提前批统招 | 120-130分 |
| 香港科技大学 | 超一本线125-140分 | 120分以上 |
| 香港理工大学 | 超一本线30-50分 | 110分以上 |
| 香港城市大学 | 超一本线30-50分 | 110分以上 |

#### 国际生赛道（占比30%）

| 课程类型 | 要求 |

| A-Level | 港大要求A*AA，其他院校要求AAA-AAB |
| IB | 38+，部分热门专业要求40+ |
| SAT/ACT | SAT 1450+或ACT 32+ |

- 可用中文应试
- 成绩受全球300+院校认可

### 语言要求

| 考试类型 | 要求 | 特殊说明 |
|---|---|---|
| 雅思 | 6.5-7.0（商科建议7.0+） | |
| 托福 | 93+（港大商学院100+） | |

- 80%录取者拥有至少1段实习或科研经历
- 热门专业建议2段以上相关实习

## 三、申请材料清单

### 核心材料

- 成绩单（中英文公证）
- 学位证明（应届生提供预毕业证明）
- 语言成绩报告
- 个人陈述（500-800字）
- 2封学术推荐信（硕士申请必需）
- 护照复印件

### 附加材料

- GMAT/GRE成绩（商科等）
- 实习证明/工作证明
- 作品集（艺术/设计/传媒类）
- 获奖证书

## 七、就业前景

- **IANG签证**：非本地生毕业后可无条件留港工作2年
- **就业行业**：IT互联网(23.9%)、金融(18.7%)、专业服务(12.6%)
- **薪资中位数**：学士学位毕业生约2.4-3.2万港币/月；金融行业约3.98万港币/月

## 八、申请策略

- 早申优势显著，第1轮录取率更高
- 避开模板化文书，突出专业匹配度与职业规划
- 香港城市大学接受六级成绩，对双非学生友好
